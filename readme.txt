== Description ==

This plugin will add an Testimonials feature inside a post or page.

Plugin Features

* Shortcode System
* 4 Different Design
* Various Options
* Custom Post Enabled. 
* Very Lightweight.
& many More.


Infinity Testimonials is an Super Easy To Use plugin that allows you to showcase the great things your customers/clients or users are saying about you or your business.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in page, post or in widgets.
4. If you want progess bar in your theme php, Place `<?php echo do_shortcode('YOUR_SHORTCODE'); ?>` in your templates
